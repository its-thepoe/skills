# @its-thepoe/codebase-content-ideas

**Agent Skill** — turn **real implementation work** into concrete content ideas: long articles, short notes, X threads, and LinkedIn posts, anchored to files and notes in the repo.

## Install into your agents

```bash
npx @its-thepoe/skills@latest install codebase-content-ideas
```

## Contents

- `SKILL.md` — workflow, modes (`mode:quick`, `mode:drafts-only`, `mode:social-only`), quality gates  
- `reference.md` — copy-paste prompt for any codebase, per-idea template
