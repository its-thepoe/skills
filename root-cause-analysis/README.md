# @its-thepoe/root-cause-analysis

**Agent Skill** — engineering RCA: mechanism, violated invariant, evidence vs inference. Optional **Fishbone (Ishikawa)**, **5 Whys**, **combined**, and **CAPA** templates—always validated, never brainstorm-only. Diagnosis first unless you ask for fixes/CAPA.

## Install into your agents

```bash
npx @its-thepoe/skills@latest install root-cause-analysis
```

## Contents

- `SKILL.md` — workflow and output shape  
- `reference.md` — theory, checklist, external references
