# @its-thepoe/aave-glass

**Agent Skill** for Aave-style refractive glass on the web.

Use it for:

- SVG `feDisplacementMap` lens effects on live DOM
- WebGL fallback for canvas and Safari video surfaces
- cross-browser glass that behaves like a lens, not generic blur

Install:

```bash
npx @its-thepoe/skills@latest install aave-glass
```
